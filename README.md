<h1>Microsoft Entra ID Tenant User Creation</h1>


<h2>Description</h2>
This lab was designed to demonstrate adding new users and groups within Azure, as well as adding user to groups.
<br />


<h2>Utilities Used</h2>

- <b>Microsoft Azure Portal</b> 



<h2>Environments Used </h2>

- <b>Windows 11</b>

<h2>Lab walk-through:</h2>

<p align="center">
On the overview page you see current user (admin) information. As well as options to add new users, manage tenants, as well as other options: <br/>
<img src="https://res.cloudinary.com/dsz8pn2ym/image/upload/v1771980721/image-8c3173f9-6ffb-4e3a-ac18-ac78f6575050_oolrut.png"/>
<br />
<br />
From your Tenant “Overview” page, click “Users”. Click “New user” and then click “Create new user”. Enter information for “User principal name”, “Display name”, and “Password”, click “Review + create”, and then click "Create. At this stage you can use the auto-generated Password created by Microsoft. To reveal the password, click the “eye” symbol. User now appears under users list:  <br/>
<img src="https://res.cloudinary.com/dsz8pn2ym/image/upload/v1771980728/image-902a9a02-db5f-43a4-ad46-21e800ff3788_urwnnd.png"/>
<br />
<br />
Now that the user is created, you can delete a user as well as restore a deleted user: <br/>
<img src="https://res.cloudinary.com/dsz8pn2ym/image/upload/v1771980731/image-a4102a27-dd99-4d62-b3ab-d3d74e18a779_tvywhu.png"/>
<br />
<br />
On the left hand side you can select "Groups". Here you create a group to add users to. You can choose group type, name, description and membership type. Once created the group will appear under "Groups" list:
<img src="https://res.cloudinary.com/dsz8pn2ym/image/upload/v1771981919/Screenshot_2026-02-24_201136_fruqmo.png"/>

<br />
<br />
Now that the group is created, once you go into it, you'll see options for the group as we'll add newly created user:
<img src="https://res.cloudinary.com/dsz8pn2ym/image/upload/v1771980750/image-31d6dfe2-de73-4f2c-b8b4-e247950d0d42_ogoy89.png"/>
